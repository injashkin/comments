Блок комментариев написан на TypeScript. Имеется возможность добавлять и удалять комментарии к статье, а также отвечать на другие комментарии. Для этого под каждым комментарием есть ссылка "Ответить". Поддерживается иерархическая структура комментариев. В качестве аватара автоматически устанавливается первая буква имени

Написал и добавил компонент для всплывающих подсказок. Демонстрацию его работы можно увидеть если навести курсором на надпись "перед тем как написать...". Подсказки адаптивные, учитывают размеры экрана. Установить подсказку можно любому элементу, для этого нужно добавить в нужный элемент атрибут data-tooltip. Можно добавлять, как простые подсказки, устанавливая атрибуту data-tooltip нужный текст, так и подсказки, структурированные HTML тегами .
